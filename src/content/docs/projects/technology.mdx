## Software applications

<CardGrid>

  <Card title = "BTAA Geoportal">

  ![](@images/geoportal-homepage.png)

  Our flagship project is a search portal that connects users to digital geospatial resources, including GIS datasets, web services, and digitized historical maps from multiple data clearinghouses and library catalogs. 

  [ <Icon name="external" class='inline-icon'/> Visit the Geoportal](https://geo.btaa.org)

  [ <Icon name="external" class='inline-icon'/> Read more about the Geoportal's key features and testimonials](/projects/geoportal).
  </Card>

  <Card title = "GBL Admin">

  ![](@images/gbl-admin.png)

  A GeoBlacklight plugin that provides a complex web-form for editing documents and an CSV-based import/export workflow for OpenGeoMetadata's Aardvark schema.

  [ <Icon name="external" class='inline-icon'/> GitHub repo](https://github.com/geobtaa/geoblacklight_admin)

  [ <Icon name="external" class='inline-icon'/> Watch a demo video](https://www.youtube.com/watch?v=lWjcr-Ow228).
  </Card>

</CardGrid>
