## Citation Field Definitions

| Category | Summary |
| --- | --- |
| Dataset Title | Provide the complete title of the dataset. |
| File Type / Format | Choose a format from the dropdown list or write in a custom value. |
| Version | Version or edition number of the data. |
| Persistent identifier or URL | Provide a static electronic location or persistent identifier (e.g., DOI or URL) used to access the dataset. |
| Author(s)/Creator(s) | Provide the name of the individual, group, or organization responsible for creating or making the data. |
| Data Publisher | Identify the organization or entity responsible for making the dataset available through archiving, publishing, or distribution. |
| Year of Publication | Indicate the year the dataset was published, which may represent a one-time or infrequent release for the cited version of the data. |
| Temporal Coverage | Time period of the dataset (free text). |
| Date accessed | Date the dataset was retrieved. If the Temporal Coverage or Year of Publication is unknown, use this field to provide some temporal context. |
